### Penda

Penda is an Open Data management system.

#### Requirements

- ``node`` version 5.11
- ``sqlite3``
- ``mongodb`` verson >2.4

#### Deployment

- ``npm install``
- ``npm install -g bower gulp``
- ``bower install``
- ``NODE_ENV=dev npm run setenv``
- ``DB=dev ./script/init.sh``
- ``npm run server``
