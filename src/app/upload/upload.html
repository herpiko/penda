Upload Requirements [ IMPORTANT ]

  • Berkas yang diterima : .csv dan .xlsx
  • Prasyarat tipe data :
    • Tipe data string dan angka akan dideteksi secara otomatis
    • Nilai untuk koordinat peta harus memiliki judul kolom lat untuk latitude dan lon untuk longatitude.
    • Nilai untuk data waktu harus memenuhi standar ISO 24 karakter, contoh 2016-06-19T02:36:41.649Z
  • Prasyarat spesifik .csv :
    • Baris pertama dari berkas adalah header atau judul kolom.
    • Delimiter / pemisah menggunakan koma (,)
    • Nilai string yang mengandung koma (,) harus dibungkus petik dua ("), contoh ... 2016, "Drs. Aminudin, S.Pd.", Jakarta ....
    • Jumlah kolom pada header dan row harus sama.
